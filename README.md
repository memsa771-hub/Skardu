# Nestopia Hotel & Resorts — Skardu

A fully interactive 3D luxury hotel website for Nestopia Hotel & Resorts, Skardu.

## Features

- **3D Virtual Tour** — Walk through hotel corridors and explore rooms in real-time (Three.js)
- **Check Availability** — Search rooms by dates and guests on the homepage
- **Full Booking System** — Multi-step booking with confirmation
- **Skardu Video Hero** — Cinematic mountain video background
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Dark Luxury Theme** — Gold & black branding matching Nestopia identity

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Pages

| Page | Path | Description |
|------|------|-------------|
| Home | `/` | Video hero, availability checker, about, location |
| 3D Tour | `/tour` | Interactive corridor & room walkthrough |
| Rooms | `/rooms` | All room types with pricing |
| Booking | `/booking` | Full reservation flow |
| Contact | `/contact` | Contact form & map |

## 3D Tour Controls

- Click to lock mouse and start walking
- **W/A/S/D** or arrow keys to move
- Mouse to look around
- Click doors to enter rooms
- **Esc** to unlock mouse
